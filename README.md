# Expense Tracker – Трекер расходов

**Автор:** Казаков Матвей

## Описание
Простое приложение для учёта личных финансов.  
Позволяет добавлять расходы, фильтровать их по категории и дате, а также считать общую сумму за выбранный период.  
Данные хранятся в JSON-файле, поэтому они не пропадут после закрытия программы.

## Как использовать
1. Запустите файл `expense_tracker.py`.
2. В верхней части введите:
   - Сумму (только положительное число)
   - Категорию (можно выбрать из списка)
   - Дату (в формате ГГГГ-ММ-ДД)
3. Нажмите «Добавить» – запись появится в таблице.
4. Чтобы удалить запись – выделите её и нажмите «Удалить выбранную запись».
5. Для фильтрации используйте блок «Фильтр»:
   - Выберите категорию
   - Укажите период (диапазон дат)
6. Внизу отображается общая сумма расходов с учётом фильтра.

## Пример (
1. Добавляем расход: 100 руб, категория «Еда», дата 2025-04-01  
2. Добавляем расход: 250 руб, категория «Транспорт», дата 2025-04-03  
3. Ставим фильтр по категории «Еда» → видим только первый расход, сумма 100 руб.  
4. Убираем фильтр, ставим период с 2025-04-01 по 2025-04-02 → сумма 100 руб.

## Заметки 
+ Категории и формат даты проверяются, так что программа не падает.
+ Данные сохраняются автоматически после каждого добавления/удаления.
+ Файл `expenses.json` создастся сам в папке со скриптом.
